# AI Dynamic Text Adventure Game

An interactive text adventure game powered by AI, allowing players to create their own characters and embark on unique adventures in various worldviews.

## ‚ú® Features

*   **10 Exquisite Themes**: Choose from various visual styles such as Dark, Cyberpunk, Fantasy, Steampunk, Neon, Nature, Ocean, Sunset, and Galaxy.
*   **Unlimited Usage**: Play as long as you want, with no usage limits, only consuming your own API quota.
*   **17 Worldviews**: Rich worlds such as Fantasy, Sci-Fi, Wuxia, Xianxia, Cthulhu Mythos, Palace Intrigue, School Mystery, Alternate History, Romance, Post-Apocalyptic, Cyberpunk, Mystery Detective, Horror, Xuanhuan, Modern Urban, Western Magic, and Transmigration, each with unique status systems.
*   **Supports Multiple AIs**: Supports all OpenAI-compatible APIs, including mainstream models like GPT-4, Claude, and Gemini, allowing you to freely choose the strongest model.
*   **Local Deployment**: Can be downloaded and run locally, giving you full control over your data and privacy.
*   **Completely Free**: All features are completely free and open, no registration required, no hidden fees.

## üöÄ Quick Start

1.  **Set up API Key**: Click the ‚öôÔ∏è Settings button in the bottom right, go to [Google AI Studio](https://aistudio.google.com/app/apikey) to apply for a free Gemini API key, then paste and save it.
2.  **Create Your Character**: Choose your favorite worldview, describe your desired character in detail, and allocate 175 attribute points (or enable cheat mode for unlimited points).
3.  **Start Adventure**: Click "Start New Adventure", and the AI will generate an exclusive opening story and initial status for you.
4.  **Enjoy the Game**: Click action options to advance the plot, or use custom actions to input any action you want to perform. You can check character status, save, or regenerate responses at any time.

## üéÆ Gameplay Tips

*   **Character Description Tips**: The more detailed your description, the more the AI-generated story will align with your imagination. Including background, personality, and goals will yield better results.
*   **Regenerate Function**: Not satisfied with the AI's response? Click the "Regenerate Answer" button, and the AI will provide a different story development.
*   **Custom Actions**: Not limited to default options! Enter any action you want to perform, and the AI will instantly respond and generate the plot.
*   **Save System**: Supports both browser saving and file download, allowing you to save your adventure progress at any time.

## ‚ùì Frequently Asked Questions

*   **Q: Is the API key paid?**
    A: Google AI Studio provides a free tier, which is generally sufficient for normal use. Payment is only required for heavy usage.
*   **Q: Is my API key safe?**
    A: The API key only exists locally in your browser and will not be uploaded to our servers, ensuring complete security.
*   **Q: Can it be used offline?**
    A: It can be downloaded and run locally, but an internet connection is still required to call the AI.
