# AI 動態文字冒險遊戲

一個由 AI 驅動的互動式文字冒險遊戲，讓玩家可以創建自己的角色，並在各種世界觀中展開獨特的冒險。

## ✨ 特色功能

*   **10 種精美主題**：深色、賽博龐克、奇幻、蒸汽龐克、霓虹、自然、海洋、夕陽、銀河等多種視覺風格任你選擇。
*   **無限制使用**：想玩多久就玩多久，沒有次數限制，只消耗你自己的 API 額度。
*   **17 種世界觀**：奇幻、科幻、武俠、仙俠、克蘇魯神話、宮鬥、校園異聞、架空歷史、戀愛、末日生存、賽博龐克、懸疑偵探、恐怖、玄幻、現代都市、西方魔法、穿越等豐富世界，每種都有獨特的狀態系統。
*   **支援多種 AI**：支援 OpenAI、Claude、Gemini 等所有相容 API，自由選擇最強模型。
*   **本地化部署**：可下載到本地運行，完全掌控你的資料和隱私。
*   **完全免費**：所有功能完全免費開放，無需註冊，無隱藏費用。

## 🚀 快速上手

1.  **設定 API 金鑰**：點擊右下角 ⚙️ 設定按鈕，前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 免費申請 Gemini API 金鑰，然後貼上並儲存。
2.  **創建你的角色**：選擇喜歡的世界觀，詳細描述你想扮演的角色，分配 175 點屬性點數（或開啟作弊模式獲得無限點數）。
3.  **開始冒險**：點擊「開始新冒險」，AI 將為你生成專屬的開場故事和初始狀態。
4.  **享受遊戲**：點擊行動選項推動劇情，或使用自訂行動輸入任何想執行的動作。隨時可查看角色狀態、存檔或重新生成回答。

## 🎮 遊戲玩法

*   **角色描述技巧**：描述越詳細，AI 生成的故事越貼合你的想像。包含背景、性格、目標等資訊效果更佳。
*   **重新生成功能**：不滿意 AI 的回應？點擊「重新生成回答」按鈕，AI 會給出不同的故事發展。
*   **自訂行動**：不侷限於預設選項！輸入任何你想執行的動作，AI 會即時回應生成劇情。
*   **存檔系統**：支援瀏覽器存檔和文件下載兩種方式，隨時保存你的冒險進度。

## ❓ 常見問題

*   **Q: API 金鑰需要付費嗎？**
    A: Google AI Studio 提供免費額度，一般使用完全足夠。只有大量使用才需要付費。
*   **Q: 我的 API 金鑰安全嗎？**
    A: 金鑰只存在你的瀏覽器本地，不會上傳到我們的服務器，完全安全。
*   **Q: 可以離線使用嗎？**
    A: 可以下載到本地運行，但呼叫 AI 時仍需要網路連線。
*   **Q: 支援哪些 AI 模型？**
    A: 支援所有 OpenAI 格式的 API，包含 GPT-4、Claude、Gemini 等主流模型。
